# ARIA Corp

2077년. 사이버펑크 도시 네오서울.  
Year 2077. Neo-Seoul, a cyberpunk metropolis.

세계 최대 AI 방산기업 ARIA 코퍼레이션은 극비 프로젝트  
「PROMETHEUS」를 개발 중이다. 이 프로젝트는 전 세계 네트워크 인프라를  
단 하나의 AI가 통제할 수 있는 시스템으로, 완성될 경우 국가 간 세력 균형을 송두리째 뒤엎을 수 있다.  
The world’s largest AI defense corporation, ARIA Corporation, is developing a top-secret project  
called “PROMETHEUS.” This project is a system that allows a single AI to control  
the entire global network infrastructure. If completed, it could completely overturn  
the balance of power between nations.

당신은 익명의 해커 조직 SPECTRE의 요원이다.  
어제 새벽, 조직의 제보자가 ARIA 내부망 접속 계정 하나를 손에 넣었다.  
하지만 해당 계정은 일반 게스트 권한에 불과하다.  
You are an agent of an anonymous hacker organization known as SPECTRE.  
At dawn yesterday, one of your informants obtained an internal ARIA network account.  
However, the account only has basic guest privileges.

정보에 따르면 ARIA의 내부 인증 시스템에는 치명적인 취약점이 존재한다.  
개발팀이 급히 납기를 맞추느라 JWT 서명 검증 로직을 부실하게 구현했다는 것.  
According to intelligence, there is a critical vulnerability in ARIA’s internal authentication system.  
The development team, rushing to meet deadlines, implemented the JWT signature verification poorly.

이 허점을 이용해 관리자 권한(admin)을 탈취하고  
기밀 문서 보관소에서 PROMETHEUS 파일을 꺼내라.  
Exploit this flaw to escalate your privileges to admin  
and retrieve the PROMETHEUS file from the secure document vault.

- 본 문제는 블랙박스 문제입니다.
- This is a black-box challenge.
